# 🎮 Atividade 01 — Acervo Inusitado

## 📌 Sobre a Atividade

Esta foi a **primeira atividade da disciplina de Frontend – 2026**.

A proposta era criar um **acervo/coleção sobre algo inusitado**, desenvolvendo um site utilizando apenas HTML e CSS.

---

## ⭐ Tema Escolhido

Escolhi criar um acervo baseado no jogo **Brawl Stars**.

O site funciona como um **museu de skins**, reunindo visuais que já passaram pelo jogo — incluindo aquelas que não aparecem mais para quem não as possui.

A ideia é preservar e organizar essas skins como uma coleção digital para fãs do jogo.

---

## 🎨 Conceito do Projeto

O visual foi inspirado em um **álbum de figurinhas**, onde cada skin representa um item colecionável.

O objetivo foi transmitir a sensação de coleção e organização, como se o usuário estivesse visualizando um álbum temático.

---

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3

O foco desta etapa foi trabalhar estrutura, organização e estilização.

---

## 🚀 Estado Atual

Atualmente o site:

* Apresenta algumas skins selecionadas
* Possui layout em formato de galeria
* É totalmente estático

---

## 🔮 Próximos Passos

Futuramente pretendo:

* Adicionar todas as skins do jogo
* Melhorar o design e a responsividade
* Tornar o site interativo
* Permitir que usuários marquem as skins que já possuem

A ideia é evoluir o projeto para algo mais completo e funcional.
