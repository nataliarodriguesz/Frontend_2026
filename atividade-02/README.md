# 🌐 Atividade 02 – Material de Estudo: Algoritmos

---

## 📚 Sobre a Atividade

Essa atividade consiste em criar um **site que funcione como material de estudo** para uma disciplina já estudada.  

- Eu escolhi **Algoritmos** como tema.  
- O **Menu Principal** apresenta o conteúdo teórico da disciplina, explicando conceitos importantes de algoritmos.  
- A **Seção de Exercícios** contém questões para treinar o que foi aprendido.  
- Atualmente, **apenas a seção de Exercícios está funcional na navegação**.  
- Todo o projeto foi desenvolvido utilizando **apenas HTML e CSS**.  

---

## 🧭 Navegação

A estrutura do site segue a lógica do repositório:

- **Menu Principal**: apresenta o conteúdo teórico sobre Algoritmos (`main.html`)  
- **Seção de Exercícios**: lista perguntas de múltipla escolha para prática, com botão “Verificar Resposta” (interativo apenas visualmente, sem backend)  

> Futuramente, a ideia é aprimorar o site, permitindo mais interatividade e funcionalidades adicionais nos exercícios.

---

## 🎨 Estrutura e Organização

- Cada exercício está dentro de uma `<div class="question">`  
- Perguntas usam `<pre>` para preservar a formatação do pseudocódigo  
- Alternativas são `<input type="radio">` agrupados por `name` para que cada questão seja independente  
- Botões de ação, como “Verificar Resposta”, usam classes/IDs estilizadas via CSS  

---

## ⚙️ Tecnologias utilizadas

- **HTML5** – Estrutura das páginas  
- **CSS3** – Estilização do site, incluindo botões, hover e formatação de pseudocódigo  

> O site segue o padrão do repositório, garantindo **consistência visual e de navegação** entre todas as atividades.
